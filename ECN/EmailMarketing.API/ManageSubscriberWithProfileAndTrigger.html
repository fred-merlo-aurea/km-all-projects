POST api/emailgroup/methods/ManageSubscriberWithProfileAndTrigger

Add one or more subscriber profiles (Email Address, Name, Custom Fields, etc.) to a single group. If the Email Address already exists the profile data will be updated. Note: To update an Email Address please use the UpdateEmailAddress method.
